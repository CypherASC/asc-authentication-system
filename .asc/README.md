# Pasta de Proteção do Sistema

Esta pasta contém arquivos essenciais para o funcionamento e proteção do sistema ASC.

## Conteúdo dos Arquivos

- `manifesto.json` - Manifesto de integridade do projeto
- `assinatura.json` - Assinatura digital para verificação
- `chave-licenca.key` - Chave de licenciamento do sistema
- `violacoes.log` - Registro de violações detectadas

## Aviso de Proteção

Esta pasta e seus arquivos são protegidos por direitos autorais e não devem ser modificados. Qualquer alteração não autorizada pode:

1. Violar os termos da licença do software
2. Causar funcionamento inadequado do sistema
3. Ser registrada automaticamente nos logs
4. Resultar em consequências legais
